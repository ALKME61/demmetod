# Типовые ошибки

## Не открывается phpMyAdmin

Проверьте контейнеры:

```bat
docker compose ps
```

Если контейнеры не запущены:

```bat
docker compose up -d
```

## Python не подключается к базе

Проверьте:

- запущен ли MySQL;
- совпадают ли логин и пароль в `db.py`;
- не занят ли порт `3306` другой базой;
- создана ли база `mosaic_demo`.

## CSV импортировался с ошибками

Частые причины:

- неверный разделитель `;` или `,`;
- осталась строка заголовков;
- дробные числа записаны через запятую;
- дата не в формате `YYYY-MM-DD`;
- ИНН сохранён как экспоненциальное число.

## Не появляются материалы после импорта

Проверьте, выполнен ли перенос из `materials_import_raw` в `materials`.

Также проверьте совпадение названий типов материалов: текст в CSV должен совпадать с текстом в таблице `material_types`.

## PyInstaller не собирает EXE

Попробуйте:

```bat
pip install --upgrade pyinstaller
```

Перед повторной сборкой закройте запущенный `.exe`.

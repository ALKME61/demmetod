# Индивидуальные приложения

Эта страница нужна для публикации общей документации без раскрытия чужих вариантов данных.

## Что выдаётся каждому студенту

Каждый студент получает свой набор исходных таблиц. Названия могут отличаться, но смысл файлов должен соответствовать пяти группам данных:

1. типы материалов;
2. типы продукции;
3. поставщики;
4. материалы;
5. связи материалов и поставщиков.

## Как добавить свои данные

1. Скопируйте `.xlsx` в папку `resources`.
2. Откройте каждый файл в Excel или LibreOffice Calc.
3. Сохраните как CSV UTF-8.
4. Удалите строку заголовков, если импорт в phpMyAdmin выполняется с ручным указанием столбцов.
5. Проверьте дробные числа, даты и ИНН.
6. Импортируйте файлы по этапу 5.

## Что нельзя смешивать

Не используйте данные из разных индивидуальных вариантов одновременно. Это может сломать связи между материалами, поставщиками и типами.
